Our Mission

Our goal is to bring Machine Learning and AI technology to Stevenson High School, which has no machine learning related classes or extracurriculars. With AI becoming increasingly relevant to every field, we believe that all high schoolers should have the oppurtunity to learn to use this technology.

Our Methods

Our club will focus on teaching techniques that are actually used in industry as well as how to publish and use machine learning models in every field using all kinds of data. We will also work on making products that could be used by Stevenson or other other students.

Our Technology

The specifics are still being worked out, but we will be using cloud computing to train and run machine learning models. We also plan on using PyTorch with Python 3. We will also be using Jupyter Notebook and Github. Prospective members do not need to know how to code to join this club.
